Please follow these instructions when importing data using the Express Insert tool
  • The First Line Must Contain the ID of the Rules where Data should be Imported
  • Data Must Start on Line 5
  • The first columns value will be used as the Label for the Reference Item
  • File Must be Saved as "tab delimited" (Mac users, please save it as Windows tab delimited)
  • Use lines 2, 3 and 4 to your liking, but keep in mind that whenever a column in Line 4 has the word "Name", that column's value will be used as the label for the item.
